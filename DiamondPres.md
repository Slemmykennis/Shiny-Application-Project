Overview
========================================================

This presentation contains the documentation for diamond price prediction application as part of the requirement to complete the final project on developing data products.

Aim
========================================================

The aim of the project is to design an application that predicts the price of diamond based on its characteristics. The characteristics that was considered in this project are: Carat, Color and Clarity.

Data
========================================================

The data used for this application is `diamonds` data set, which is part of `ggplot2` package.
This data set contains the information about 53940 diamonds with 10 variables:
